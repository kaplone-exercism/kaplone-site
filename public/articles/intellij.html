Environnements : IntelliJ

Bien qu'étant un IDE payant sous licence propriétaire, IntelliJ est aussi proposé gratuitement en version communautaire.

jetbrains, l'éditeur du programme propose une gamme complète d'IDE spécifiques à des nombreux langages :

  • Java -> IntelliJ Idea
  • Python -> PyCharm
  • Ruby -> RubyMine
  • JS -> WebStrom
  • PHP -> PhpStorm
  • C/C++ -> CLion
  • IOS/OSX -> AppCode
  • GoLang -> GogLand
    • ...

En faisant ce choix, chaque produit est taillé au plus juste de chaque besoin.
On retouve aussi une continuité en passant d'un langage à l'autre.

En 2014, l'environnement 'officiel' de développement Android s'éloigne d'Eclipse.
On voit apparaitre Android Studio, basé sur IntelliJ Idea.
