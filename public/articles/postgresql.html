Bases de données : PostgreSql

PostgreSql est un SGBD relationnel open source.

J'ai peu d'expérience avec les bases de données commerciales (Oracle, SQL server, ...) et étrangement aussi avec Mysql / MariaDB.
Pour la persistances des données en mode relationnel, j'ai surtout travaillé avec PostgreSQL.

Du coup, je connais plus cet écosystème là : psql, PGadmin, PostGis.

J'ai toujours été convaincu par la maturité de Postgres.
La facilité de mise en oeuvre et la souplesse de MongoDB faisait souvent pencher la balance vers cette solution

Depuis quelques temps cependant les cartes sont rebattues :
des solutions sont apparues, permettant d'introduire une partie de données sans schéma, basées sur des documents Json dans lesquels on peut exécuter des requêtes avec une syntaxe spécifique.
La solution pourrait se révéler très séduisante ...
