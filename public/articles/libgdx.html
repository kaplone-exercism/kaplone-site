Frameworks : LibGDX

LibGDX est un framework pour le développement en java de jeux et d'applications graphiques multi-OS.

LibGDX est construit sur Lightweight Java Game Library (LWJGL) qui lui fournit tous les supports multimédias (OpenGL, OpenAL, ...).
L'idée est ensuite de séparer le code en 3 bases :

  • Une partie contenant les fichiers spécifiques à LibGDX.
  • Autant de parties "launcher" que de supports de déploiement souhaités.
  • La partie contenant l'application elle même.
