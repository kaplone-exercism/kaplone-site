Langages : Java

Java est un des langages les plus utilisés au monde.

Depuis 2012 j'ai développé en Java des applications serveur, desktop et android.

Pour les serveur je me suis appuyé sur les frameworks Play! et Spark-java. J'ai commencé a regarder le fonctionnent de J2EE (servlets et jsp) à partir de 2017 au CNAM.

Pour les application desktop, certaines s'exécutent en ligne de commande, mais la plupart on une interface écrite en JavaFX.
J'ai aussi pu développer un projet avec le framework LibGDX qui permet une orientation plus évènementielle et multimédia.

Les projets pour android ont été développés avec le toolkit Android, et avec Gluon-mobile qui s'appuie sur JavaFX.

    Dès la sortie de java 8 je me suis intéressé aux apports de cette version.
    de fait, les apport sont importants :
  • introduction des lambdas expressions :
    Cet ajout a été possible grace à l'introduction des interfaces fonctionnelles, des interfaces ayant une seules méthode abstraite.
    On peut aussi noter l'ajout de la possibilité d'avoir des méthodes définies dans les interfaces, des méthodes par défaut.
    L'utilisation des lambdas expressions se substitue souvent aventageusement à l'écriture d'une classe anonyme ...
  • introduction des objets Function, permettant de définir des fonction de premier ordre.
  • introduction des streams, des plus infinis et imutables, évaluables de façon paresseuse et/ou parallèle.
  • introduction d'une nouvelle API pour la gestion des dates et des durées.

    • Je garde un oeil sur Java 9 dont la sortie est annoncée pour l'été 2017.
