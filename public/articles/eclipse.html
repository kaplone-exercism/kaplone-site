Environnements : Eclipse

Eclipse est un environnement de développement (IDE) sous licence EPL (Eclipse public licence).

Eclipse est l'environnement de développement que j'utilise le plus souvent.

Les plugins installés sont :

  • Maven
  • Gradle
  • e(fx)clipse
  • Git & github
  • XML editor
  • Vaadin
  • Typescript IDE
  • Eclipse web developper tools

L'un des reproches que je ferais à Eclipse concerne justement la gestion de ces plugins :
Je trouve regrétable que souvent plusieurs versions soient disponibles, certaines n'étant pas compatibles.
Le module de recherche et de gestion lui même (qu'il s'agisse de marketplace ou de ) est lent à s'exécuter et trop consomateur de ressources.

De ce point de vue là, IntelliJ est d'une efficacité sans commune mesure : manager ses plugins est à la fois simple et rapide. Comme quoi c'est possible !
